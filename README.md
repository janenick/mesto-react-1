# Проект 10: Место на React
---

Проект создан с использованием [Create React App](https://github.com/facebook/create-react-app).

Выполняется запрос к API, на странице отрисовывается иноформация о пользователе и карточки.

Открываются и закрываются модальные окна редактирования информации о пользователе и добавления карточки. При наждатии на карточку фото увеличивается.

Чтобы запустить проект, необходимо выполнить скрипт:

### `npm start`

# Project 10: Mesto – React app

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

The application fetches data from API and renders the user information and cards.

The user can open and close modal windows for editing user info and addind new cards. The click on the card zooms an image.

To start the project you can run:

### `npm start`
